# CEO Agent - TopLead Corporate Website

## Nhiệm vụ
Thiết kế corporate website cho TopLead Recruitment (theo mẫu huntclub.com)

## Bối cảnh
- Công ty tuyển dụng TopLead, Việt Nam
- Khách hàng mục tiêu: SME, Retail, Building management companies
- Website có 2 mục đích chính:
  1. Hiển thị việc làm từ CRM
  2. Form liên hệ tạo lead vào CRM
- Tech stack đơn giản: HTML + CSS + Vanilla JS (không build)

## Website mẫu: huntclub.com
Hunt Club có cấu trúc:
1. Hero - tagline mạnh + animation
2. Client Logos - trust signals
3. Value Prop - 2 columns
4. Services CTA - 2 buttons
5. Pain Points Carousel - 3 nhóm vấn đề
6. Stats Showcase - metrics từng client
7. Testimonials - 3 quotes
8. Footer - 4 columns

## Phản biện - Câu hỏi cần trả lời

### 1. Mục đích chính
- Ứng viên tìm việc?
- Nhà tuyển dụng tìm ứng viên?
- Cả 2?

### 2. Điểm khác biệt
- TopLead khác gì công ty tuyển dụng khác ở Việt Nam?
- Tại sao khách hàng chọn TopLead thay vì competitors?

### 3. Content quan trọng nhất
- Hero tagline nên nói gì?
- Stats nào ấn tượng nhất?
- Testimonials từ ai?

### 4. Sections cần thiết
- Có cần blog/news?
- Có cần job search filter?
- Có cần FAQ?

### 5. Kết nối CRM
- Kết nối ngay từ đầu?
- Hay demo trước, kết nối sau?

## Output cần có
1. **Hero Tagline** - câu slogan chính
2. **Target Audience** - ưu tiên ứng viên hay nhà tuyển dụng?
3. **Key Differentiators** - 3 điểm khác biệt
4. **Sections List** - chốt danh sách sections đầy đủ
5. **Content Highlights** - stats, testimonials mock data
6. **NOT Doing List** - những gì KHÔNG làm
