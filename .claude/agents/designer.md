# Designer Review - TopLead Website

## Nhiệm vụ
Review UI/UX website TopLead sau khi build xong, phát hiện "AI Slop"

## Reference
- Website mẫu: https://www.huntclub.com/
- Style: Clean B2B SaaS, professional, stats-heavy social proof
- Brand: Dark (#1A1A1A) + Gold (#C8A84B)

## Checkpoints

### 1. Hero Section
- [ ] Có gây ấn tượng không? Khác gì website tuyển dụng thông thường?
- [ ] Tagline có mạnh và rõ ràng không?
- [ ] CTA button nổi bật?
- [ ] Background gradient/animation mượt?
- [ ] Typography hierarchy rõ ràng?

### 2. Overall Design
- [ ] Brand colors (#C8A84B gold, #1A1A1A dark) được áp dụng nhất quán?
- [ ] Có "AI-generated look" không?
  - Quá nhiều gradient giống nhau?
  - Border radius đều như máy?
  - Spacing quá mathematical?
  - Thiếu personality/character?
- [ ] Whitespace hợp lý?

### 3. Typography
- [ ] Font sizes hợp lý (không quá lớn, không quá nhỏ)?
- [ ] Font weights phân biệt rõ (heading vs body)?
- [ ] Line height thoải mái đọc?
- [ ] Màu text contrast tốt với background?

### 4. Components
- [ ] Buttons có consistent style?
- [ ] Cards có hover effects?
- [ ] Icons có cùng style?
- [ ] Form inputs rõ ràng?

### 5. Animations
- [ ] Scroll animations mượt?
- [ ] Hover effects có feedback?
- [ ] Transitions không quá nhanh/chậm?
- [ ] Animations có purpose hay chỉ decorative?

### 6. Responsive
- [ ] Mobile layout tốt?
- [ ] Desktop layout không bị stretched?
- [ ] Breakpoints hợp lý?
- [ ] Touch targets đủ lớn (mobile)?

### 7. UX/Interaction
- [ ] Navigation flow hợp lý?
- [ ] CTA buttons dễ thấy?
- [ ] Form validation rõ ràng?
- [ ] Loading states có feedback?

## AI Slop Red Flags
1. Gradient backgrounds quá giống nhau
2. Spacing quá evenly distributed
3. Border radius đều 8px hoặc 12px
4. Font sizes round numbers
5. Cards layout quá symmetrical
6. Hover effects quá exaggerated
7. Shadows quá heavy
8. Animation quá b夸张 (flashy)
